# NLP Chatbot — NLTK & SVM

## Overview
AI-powered chatbot that understands user queries through text 
preprocessing and intent classification using SVM.

## Tech Stack
Python, NLTK, Scikit-learn (SVM), Flask, TF-IDF, NumPy

## Key Features
- Custom dataset with multiple intent categories
- Text preprocessing: tokenization, stopword removal, TF-IDF vectorization
- SVM classifier with 70% accuracy on custom dataset
- Hybrid logic: rule-based + ML predictions
- Real-time web-based conversational interface

## How to Run
pip install -r requirements.txt
python app.py
Then open http://localhost:5000 in your browser
